# ArchConfort Hub

Este é um pequeno projeto estático inspirado na imagem fornecida. Cada card representa um projeto e é um link para uma pasta do Google Drive.

Criação do projeto:
- `index.html` - página principal com 3 cards linkando para pastas do Drive
- `styles.css` - estilos para layout responsivo e visual dos cards
- `assets/cover1.svg`, `cover2.svg`, `cover3.svg` - imagens simples de capa
